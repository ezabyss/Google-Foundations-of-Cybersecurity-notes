# Screenshots – Apply Filters to SQL Queries

This directory contains screenshots captured during the execution of the SQL lab **“Apply Filters to SQL Queries”**, completed using the MariaDB shell.

These screenshots provide **visual evidence** of applying SQL filters to retrieve specific security-relevant information from the `organization` database.

---

## Screenshot Checklist and Filenames

### Task 1: List All Organization Machines
- **task-01-select-machines-os.png**  
  Shows query selecting `device_id` and `operating_system` from the `machines` table

---

### Task 2: Retrieve Machines with OS 2
- **task-02-filter-os2-machines.png**  
  Shows use of the `WHERE` clause to filter machines running **OS 2**

---

### Task 3: List Employees in Specific Departments
- **task-03-filter-finance-department.png**  
  Shows filtering employees in the **Finance** department

- **task-03-filter-sales-department.png**  
  Shows filtering employees in the **Sales** department

---

### Task 4: Identify Employee Machines
- **task-04-filter-specific-office.png**  
  Shows query identifying the employee assigned to office `South-109`

- **task-04-like-south-building.png**  
  Shows use of the `LIKE` operator to retrieve all employees in the **South** building
